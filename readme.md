# Readme

## Context

I've just testing how QT works.

## Requirements
- Python 3.9
- Internet connection
- [OpenAI API token](https://platform.openai.com/account/api-keys)

## Install dependencies
Execute:
`pip install -r requirements.txt`

## Run the project
Copy `.env.sample` to `.env`
Add your ChatGPT token to `.env` file

Execute:
`python3 main.py`
